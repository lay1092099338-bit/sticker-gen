**API endpoints:**
- `POST /api/parse-excel` — multer file upload → XLSX parse → JSON
- `POST /api/generate-prompt` — builds prompt from copywriting+theme+style
- `POST /api/generate-image` — proxies to Modelverse image gen, parses base64 from markdown `![image](data:...)` format
- `POST /api/refine-prompt` — takes existing prompt + user requirements, calls Modelverse text LLM to rewrite prompt, returns refined version

**Prompt structure:**
- Round sticker, 5x5cm, circular, white border
- Style description (from styleMap)
- Center text = copywriting column
- Theme element = theme column
- Baby shower product positioning
- 300dpi, professional

**Key technical notes:**
- Modelverse image response comes back as markdown: `![image](data:image/jpeg;base64,...)`
- Parse with regex: `/!\[.*?\]\((data:image\/[^;]+;base64,[^)]+)\)/`
- API key is stored server-side (not exposed to frontend); frontend API key input is optional override
- User's own Modelverse key had no image generation permissions → switched to server-side key
